= Spiral Order

Given a matrix of m x n elements (m rows, n columns), return all elements of the matrix in spiral order.

== Input Format:
Matrix

== Output Format
Array of numbers

== Sample Input
```
[
  [ 1, 2, 3 ],
  [ 4, 5, 6 ],
  [ 7, 8, 9 ]
]
```


== Sample Output
```
[1, 2, 3, 6, 9, 8, 7, 4, 5]
```
